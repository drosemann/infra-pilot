# branding guidelines: cosmic infra (infra pilot)

overview
- cosmic infra is the branding identity for infra pilot. it emphasizes depth, control, and scalable infrastructure with a space-inspired aesthetic.

color & typography
- primary: #6C5CE7
- secondary: #EC4899
- accent: #22D3EE
- background: #0B0720
- text: #F6F3FF
- typography: inter (or system-ui) with a 1.0 base, scale for headings

logo variants
- default: branding/assets/logo.svg
- alt: branding/assets/logo-alt.svg
- a simple selector is provided in the management panel to switch variants at runtime.

usage rules
- use brand.primary for primary actions and headers.
- use brand.accent for interactive highlights and focus states.
- maintain high contrast on all surfaces; ensure accessibility compliance (wcag aa).
- the orbit ring should appear around the logo in branded contexts.

ui kit & components
- brandbutton, brandcard, brandinput reuse brand tokens.
- navbars, tooltips, and toasts adopt the brand color semantics.

asset packaging
- all branding assets should be placed under branding/ (logo.svg, logo-alt.svg, etc.). the branding.md in branding/ documents usage.

versioning
- version 1.0

# Feature 55: Live Spectate / OBS Integration

## Overview
OBS-studio plugin for auto scene-switching based on game state. Spectator slots, delay config, stream proxy.

## Components
- **Integration Service Module**: `gaming/live_spectate.py` - Spectate service
- **Management Panel Page**: `gaming/LiveSpectate.tsx` - Spectate UI

## Features
- OBS WebSocket integration
- Auto scene-switching
- Spectator slot management
- Configurable stream delay
- Stream proxy
- Multi-camera support
- Overlay management
- Replay buffer
- Live stats overlay
- Integration with tournament manager

# Feature 56: Game Analytics Dashboard

## Overview
Per-game metrics: active players, session duration, peak concurrency, revenue per player. Heatmaps of popular times/maps.

## Components
- **Management Panel Page**: `gaming/GameAnalytics.tsx` - Analytics dashboard
- **Integration Service Module**: `gaming/game_analytics.py` - Analytics data service

## Metrics Tracked
- Active players (real-time)
- Session duration distribution
- Peak concurrency
- Revenue per player (ARPPU)
- Player retention
- Churn rate
- Map popularity
- Time-of-day heatmaps
- Geography distribution
- Server performance correlation

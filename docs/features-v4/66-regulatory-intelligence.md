# Feature 66: Regulatory Intelligence Engine

## Overview
Automated regulatory change monitoring and impact analysis. Monitors government and standards sources for regulatory updates affecting compliance frameworks.

## Components
- **Integration Module**: `RegulatoryIntelligenceEngine` — change detection, source management, impact analysis
- **Orchestrator Cog**: `regulatory_intel_cog.py`
- **Management Panel**: `RegulatoryIntel.tsx`
- **CLI Commands**: `regintel changes|detect|sources|stats`

## Key Features
- Regulatory change monitoring
- Impact level classification
- Source management (government, regulatory, standards)
- Framework impact mapping
